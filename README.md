# hibot-flask-guestbook

> Your first database. A guestbook app with Flask and SQLite — learn how to store and retrieve data.

Built with **Flask + SQLite**.

## Quick Start

1. Clone: `git clone https://github.com/Cartooli/hibot-flask-guestbook.git`
2. Install: `pip install -r requirements.txt`
3. Run: `python app.py`
4. Visit: `http://localhost:5000`
5. Add messages and watch them persist in the database

## What You'll Learn

- Core concepts of Flask + SQLite
- How hibot-flask-guestbook is structured
- How to customize and deploy your own version

## See It Live

Open the demo and see it in action, then rebuild it your way on [code.hibot.space](https://code.hibot.space/edtech-tutorials.html).
