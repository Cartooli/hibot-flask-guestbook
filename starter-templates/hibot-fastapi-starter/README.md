# hibot-fastapi-starter

> Your first backend server. A text processing API built with FastAPI — see how routes, requests, and responses work.

Built with **FastAPI (Python)**.

## Quick Start

1. Clone: `git clone https://github.com/Cartooli/hibot-fastapi-starter.git`
2. Install: `pip install -r requirements.txt`
3. Run: `python main.py` (or `fastapi run main.py`)
4. Visit: `http://localhost:8000/docs` to see the API
5. Customize: Edit endpoints and add your own routes

## What You'll Learn

- Core concepts of FastAPI (Python)
- How hibot-fastapi-starter is structured
- How to customize and deploy your own version

## See It Live

Open the demo and see it in action, then rebuild it your way on [code.hibot.space](https://code.hibot.space/edtech-tutorials.html).
